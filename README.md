# Endoease
🌸 EndoEase — Endometriosis Health Companion

A frontend health tracking web application designed 
for endometriosis patients to monitor symptoms, 
track cycles, and manage their health journey.

## Features
- 🗺️ Interactive SVG pain body map
- 📊 Symptom & cycle tracking across 6 parameters
- 🥗 7-day personalized meal planner
- 💊 Medication adherence checklist
- 📈 Chart.js trend visualizations
- 📔 Mood journal & trigger diary
- 🌿 Exercise & nutrition recommendations

## Tech Stack
HTML5 · CSS3 · Vanilla JavaScript · Chart.js · SVG · Responsive Design
